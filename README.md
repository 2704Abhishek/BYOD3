# BYOD 3 – CI/CD Pipeline Setup & Infrastructure Planning

## Objective
This project demonstrates an automated CI/CD pipeline using Jenkins, GitHub Webhooks, ngrok, and Terraform.

## Tasks Covered
- Automated Jenkins triggering via GitHub Webhook
- Secure credential injection
- Branch-specific Terraform planning
- Manual approval gate for dev environment

## Tools Used
- Jenkins
- GitHub
- ngrok
- Terraform
- AWS

## Branch Strategy
- dev → requires manual approval
- main → auto plan execution
